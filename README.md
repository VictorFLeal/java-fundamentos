# ☕ java-fundamentos

Repositório de exercícios práticos de Java desenvolvidos como parte dos meus estudos na trilha **Maratona Java** do [DevDojo](https://devdojo.academy/).

O objetivo é praticar os conceitos aprendidos no curso com exercícios de contexto real — do básico ao avançado — e evoluir com autonomia crescente, sem depender de IA para resolver os problemas.

---

## 📁 Estrutura do Projeto

```
src/
└── main/
    └── java/
        └── com/
            └── victordev/
                ├── DIA1/       # Condicionais, loops, break, continue
                │   ├── ATV01
                │   ├── ATV02
                │   ├── ATV03
                │   ├── ATV04
                │   └── ATV05
                ├── DIA2/       # ...
                │   ├── ATV01
                │   └── ...
                └── ...
```

Cada pasta `DIA{N}` representa um dia de estudo com até 5 atividades (`ATV01` a `ATV05`).  
O projeto cresce conforme o avanço no curso.

---

## 🧠 Conteúdo por Fase

| Fase | Conteúdo |
|------|----------|
| DIA1 | Condicionais (if/else, switch), loops (for, while, do-while), break e continue |
| DIA2 | Arrays *(em breve)* |
| ... | *(evolui conforme o curso)* |
